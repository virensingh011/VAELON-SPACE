# VAELON SPACE Website

An aerospace-themed company profile website for **VAELON SPACE**, built as a single-page HTML/CSS/JS project with local image assets, circular logo branding, propulsion visuals, mission-style sections, and a lightweight Node.js local server.

## Preview

Run the local server and open:

```bash
http://localhost:8020
```

## Project Files

```text
vaelon-aerospace/
├── index.html
├── server.js
├── start-localhost.bat
├── README.md
└── assets/
    ├── vaelon-logo-enhanced.png
```

## Features

- Circular VAELON SPACE logo badge
- Aerospace hero section with Hall thruster imagery
- Propulsion, systems, programs, performance, leadership, reports, and contact sections
- Multiple image assets used across the website without repeating the same picture everywhere
- Responsive layout for desktop and mobile
- No build tools required
- Simple local Node.js server included

## How To Run

### Option 1: Windows Launcher

Double-click:

```text
start-localhost.bat
```

Then open:

```text
http://localhost:8020
```

Keep the terminal window open while viewing the website.

### Option 2: Node.js Command

```bash
set PORT=8020
node server.js
```

Then open:

```text
http://localhost:8020
```

## GitHub Upload Steps

1. Create a new GitHub repository.
2. Upload all files inside this folder.
3. Make sure the `assets` folder is included.
4. Keep `index.html` in the root of the repository.
5. Add this README as the project description page.
